# Gimmick 📱

App mobile per la cattura rapida di contenuti multi-formato con assistente AI.

## Cosa fa

- **Cattura** → Foto, video, testo, voce, file, galleria
- **Chat AI** → Comandi vocali e testuali
- **Teleport** → Invia contenuti ad altre app (es. Magicaboola)

## Stack

- **Mobile**: Expo + React Native + TypeScript + Zustand
- **Backend**: Express.js + Supabase

## Quick Start

```bash
# Backend
cd backend && npm install && npm run dev

# Mobile
cd mobile && npm install && npx expo start
```
